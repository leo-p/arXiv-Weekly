# arXiv-Weekly
Weekly newsletter of the best arXiv papers for Computer Science.

## Setup python environment
```zsh
$ pipenv install
$ source credentials.sh
```

## Download papers from arXiv Sanity Preserver
```zsh
$ pipenv run python get_weekly_papers.py
```
